Los preferidos de nuestro staff

Que nuestros lectores y lectoras aman

  • Introducción a los algoritmos

    El nombre de este libro se explica por sí solo; ¡es una buena Introducción a los algoritmos! Introducción a los algoritmos, o CLRS, en referencia a los apellidos de los autores, profundiza y abarca una serie de algoritmos a lo largo de varios capítulos independientes.

    Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest, Clifford Stein

  • La estructura e interpretación de los programas computacionales (SICP)

    Este es uno de los mejores libros para aprender los conceptos básicos de la programación. Incluido por el MIT en uno de sus cursos básicos de programación, SICP es un libro de programación de carácter general que utiliza Scheme para ilustrar diversos conceptos de programación.

    Harold Abelson, Gerald Jay Sussman, Julie Sussman

  • El programador limpio: un código de conducta para programadores profesionales

    Compilado por el experimentado ingeniero de software y autor Robert C. Martin (también conocido como "Tío Bob"), este libro abarca las técnicas y herramientas de la verdadera artesanía de software. Explica cómo escribir código limpio y muestra cómo cultivar la actitud de un programador experto y profesional.

    Steve McConnell

  • Código completo: un manual práctico de construcción de software

    ¿Te gustaría saber cómo escribir un buen código al margen de la arquitectura del lenguaje de programación que hayas elegido? Después, podrías leer Code Complete: A Practical Handbook of Software Construction ("Código completo: un práctico manual sobre creación de software"). Este aborda de manera exhaustiva todo lo relacionado con la buena estructura de código.

    Steve McConnell

  • Aprendiendo JavaScript

    una excelente guía para aquellos interesados en aprender o mejorar sus habilidades en JavaScript, Carlos Azaustre, con su experiencia en el desarrollo web, hace que el contenido sea accesible y práctico, facilitando el aprendizaje a través de ejemplos claros y detallados. Es una obra muy recomendada para quienes desean entender y dominar JavaScript.

    Carlos Azaustre

  • Clean Code

    Cada año, se invierten innumerables horas y se pierden numerosos recursos debido a código mal escrito, ralentizando el desarrollo, disminuyendo la productividad, generando graves fallos e incluso pudiendo acabar con la organización o empresa. El reconocido experto de software Robert C. Martin, junto con sus colegas de Object Mentor, nos presentan sus óptimas técnicas y metodologías ágiles para limpiar el código sobre la marcha y crearlo de forma correcta, de este modo mejorará como programador.

    Robert C. Martin

  • Eloquent JavaScript

    Este es un libro acerca de instruir computadoras. Hoy en dia las computadoras son tan comunes como los destornilladores (aunque bastante más complejas que estos), y hacer que hagan exactamente lo que quieres que hagan no siempre es fácil. Si la tarea que tienes para tu computadora es común, y bien entendida, tal y como mostrarte tu correo electrónico o funcionar como una calculadora, puedes abrir la aplicación apropiada y ponerte a trabajar en ella. Pero para realizar tareas únicas o abiertas, es posible que no haya una aplicación disponible.

    Marijn Haverbeke

  • JavaScript and jQuery: The Missing Manual

    la Web era un lugar bastante aburrido en sus primeros días. Se construyeron páginas web de HTML simple, para que pudieran mostrar información, y eso era todo. La gente hacía clic en un enlace y luego esperaba a que se cargara una nueva página web. Eso fue Lo más interactivo que hay. Hoy en día, la mayoría de los sitios web son casi tan sensibles como los programas de un ordenador de sobremesa ordenador, reaccionando inmediatamente a cada clic del ratón. Y todo gracias a la temas de este libro: JavaScript y su compañero, jQuery

    David McFarland
